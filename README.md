# Neverours DM Ticket System

A Discord bot that creates private support tickets from a panel button and forwards messages between user DMs and ticket channels.

## Features
- Ticket panel with buttons (`Create Ticket`, `Clear DM`)
- Private ticket channel creation with role-based permissions
- Two-way message forwarding (DM <-> ticket channel)
- Ticket close action with user notification
- Basic log messages for ticket actions

## Setup
1. Install dependencies:
   - `npm install`
2. Configure your `.env` values.
3. Start the bot:
   - `npm start`
